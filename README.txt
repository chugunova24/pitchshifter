Hello!

Description:
    This project is the implementation of the Guitar Pitch Shifter algorithm.

Link to the original article:
    https://www.guitarpitchshifter.com/algorithm.html

Follow the next command in the terminal to start the program:
    "./run.sh <input.wav> <output.wav> <time_stretch_ratio>",
    where <input.wav> - is path/to/file
          <output.wav> - is name of output file
          <time_stretch_ratio> - the parameter of the algorithm
